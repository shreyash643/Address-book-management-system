# 📒 Address Book Management System (C)

## 📌 Project Title
**Address Book**

## 🧾 Project Brief
The Address Book is a simple project developed in **C** for efficiently managing contact information. It allows users to **add, search, update, and delete contacts**, and **view the contact list in sorted order**.

All contact information, including **name, phone number, and email**, is stored using **structures**, and the data is saved in a **file**, ensuring that contacts remain available even after the program is closed.

---

## 🚀 Features
- Add new contacts  
- Search existing contacts  
- Update contact details  
- Delete contacts  
- Display all contacts (sorted)  
- Persistent storage using file handling  

---

## 🛠 Technologies Used
- C Programming Language  
- Structures  
- Arrays  
- Functions  
- File Handling  
- Strings  

---

## ⚙️ How It Works
- Contacts are stored using **structures**  
- Data is read from and written to a **file**  
- User interacts through a **menu-driven interface**  
- Sorting is applied when displaying contacts  

---

## 🧩 Key Challenges & Learnings
- Learned how to read and store data from files using structures  
- Improved logical thinking while validating user inputs  
- Understood how small conditions impact overall program behavior  
- Implemented **email validation** ensuring correct format and uniqueness
